Selection Sort 选择排序

复杂度

时间

Best: \( \Omega(n^2) \), Average: \( \Theta(n^2) \), Worst: \( O(n^2) \)

空间

\( O(1) \)

核心实现

两层循环，找到并记录其中最小的元素位置，最后跟已排序好元素的Next元素做个替换。

代码

#include "selection_sort.h"
#include <utility>

namespace algotest {
  /**
   * avg: O(n^2)
   */
  void selection_sort(std::vector<uint32_t>& vec) {
    const auto len = vec.size();
    if (len < 2) {
      return;
    }

    const auto last = len - 1;
    for (size_t i = 0; i < last; ++i) {
      auto imin = i; // 记录的最小值的下标
      for (auto j = i + 1; j < len; ++j) {
        if (vec[j] < vec[imin]) {
          imin = j;
        }
      }
      if (i != imin) {
        std::swap(vec[i], vec[imin]);
      }
    }
  }
}

总结

一样没什么卵用的暴力算法，一般能出现使用场景的也会被insertion sort所代替，所以说，这个算法真的是一点卵用也没有。
